# Print method for fullfit results

Displays a summary header with outcome, model type, method, and number
of multivariable predictors before printing the results table.

## Usage

``` r
# S3 method for class 'fullfit_result'
print(x, ...)
```

## Arguments

- x:

  fullfit_result object.

- ...:

  Additional arguments passed to print methods.

## Value

Invisibly returns the input object `x`. Called for its side effect of
printing a formatted summary to the console.
